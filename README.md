# Props Trading Landing Page

A modern, responsive landing page for a props trading website built with React and Vite.

## Features

- Modern navy + dark blue + white design
- Responsive navbar with login link to app.plip.xyz
- Hero section with call-to-action buttons
- Interactive FAQ section
- Footer with social links and navigation

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- CSS3 (Custom Properties)
